
## What’s New in 4.2.1

The Light V2 Proxy was modified in conjunction with O.S. Release 4.2.1. This included:

### Capabilities
The [advanced\_scene\_support ][1]capability has been enhanced to support eh ability of being set dynamically.

The ability to hide properties for a driver in Composer Pro is available through the dynamic capability: hide\_proxy\_properties.

The ability to hide the Events interface for a driver in Composer Pro is available through the dynamic capability: hide\_proxy\_events.

The ability to hide the Commands interface for a driver in Composer Pro is available through the dynamic capability: hide\_proxy\_commands.

The ability to hide the programming conditionals and loops for a driver in Composer Pro is available through the dynamic capability: hide\_proxy\_conditionals\_loops.


[1]:	https://snap-one.github.io/docs-driverworks-proxyprotocol-lightv2/#advanced-lighting-scene-agent-capabilities
