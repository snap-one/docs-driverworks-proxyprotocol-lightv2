## hide\_proxy\_events

Enabling this capability prevents the Events interface in Composer Pro programming from being displayed for a single bulb driver.  While rarely used, this property is useful when configuring smart bulb hubs where only one light serves as a controllable/configurable hub entity in the project. This capability can be changed through a `DYNAMIC_CAPABILITIES_CHANGED`  notification.


### Signature

`<hide_proxy_events></hide_proxy_events>`


### Type

Boolean. 


### Dynamic Capability

Yes


### Example

```xml
<capabilities>
    <hide_proerty_events>true</hide_proerty_events>
</capabilities>
```
