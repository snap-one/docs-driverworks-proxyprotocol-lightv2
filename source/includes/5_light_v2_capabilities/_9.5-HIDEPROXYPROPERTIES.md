## hide\_proxy\_properties

Enabling this capability prevents the properties from an array of lights from being displayed in ComposerPro. While rarely used, this property is useful when configuring smart bulb hubs where only one light serves as a controllable/configurable entity in the project. This capability can be changed through a `DYNAMIC_CAPABILITIES_CHANGED`  notification.


### Signature

`<hide_proxy_properties></<hide_proxy_properties>`


### Type

Boolean


### Dynamic Capability

Yes


### Example

```xml
<capabilities>
    <hide_proxy_properties>True</hide_proxy_properties>
</capabilities>
```
